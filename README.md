📊 Startup Growth & Funding Analysis (Excel Dashboard)
Project Description

This project explores startup ecosystem trends, focusing on funding, revenue, industry performance, and regional growth.

Objectives;

Analyze funding vs revenue,
Identify top-performing industries,
Evaluate regional performance

Tools Used;
Excel,
Pivot Tables,
Charts & Dashboard

Key Insights;
FinTech attracts the most funding,
E-Commerce generates the most revenue,
Europe leads in performance,
Startup activity peaked in recent years

Recommendations;
Focus on scalable industries,
Optimize funding allocation,
Expand in high-performing regions
